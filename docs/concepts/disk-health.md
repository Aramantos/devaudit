# Disk Health

## What is it?

Disk health monitoring tracks your storage space and drive condition. Running out of space can prevent security updates and cause system instability.

## Why it matters

Low disk space prevents security patches from installing, can corrupt files, and makes systems vulnerable to attacks that exploit outdated software.

## When to update

Free up space when any drive reaches 85% full. Critical action needed at 95% full.

## When to skip

Never ignore disk space warnings—they directly impact security update capability.

## How to fix

Delete temporary files, old downloads, unneeded applications. Move large files to external storage. Run disk cleanup utilities.

## Risks

Full disks prevent security updates, cause data corruption, system crashes, and can brick systems during failed updates.
