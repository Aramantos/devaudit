# Backups

## What is it?

Backups create copies of your data stored separately from your main system, protecting against hardware failure, ransomware, and accidental deletion.

## Why it matters

Without backups, ransomware can permanently encrypt your data. Hardware failures, theft, or user errors can cause total data loss.

## When to update

Configure automated backups immediately if none exist. Ensure backups run at least weekly for critical data.

## When to skip

Never skip backups—data loss is permanent and often unrecoverable.

## How to fix

- **Windows**: Enable File History in Settings → Update & Security → Backup
- **macOS**: Turn on Time Machine in System Preferences → Time Machine
- **Linux**: Use rsnapshot, timeshift, or borg for automated backups

Follow the 3-2-1 rule: 3 copies of data, on 2 different media types, with 1 copy offsite.

## Risks

No backups means total data loss from ransomware, hardware failure, theft, fire, or user error. Businesses can fold; personal data (photos, documents) lost forever.
