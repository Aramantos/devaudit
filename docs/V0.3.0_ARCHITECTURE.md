# DevAudit v0.3.0 Architecture Plan: System-Wide Scanning

**Version:** 0.3.0
**Theme:** Beyond Packages - Full System Security Auditing
**Target:** February 2025
**Status:** Planning Phase

---

## Executive Summary

v0.3.0 represents DevAudit's evolution from a developer-focused package auditor to a **comprehensive personal security assistant**. This release adds 8 new system-wide auditors that scan BIOS, OS updates, antivirus status, drivers, disk health, backups, encryption, and firewalls.

**Key Goals:**
1. Audit entire system, not just development tools
2. Provide educational context for every finding
3. Maintain cross-platform compatibility (Windows, macOS, Linux)
4. Keep privacy-first architecture (100% local)
5. Prepare foundation for v0.4.0 remediation engine

## New Auditor Specifications

### 1. BIOS/UEFI Auditor (`bios_audit.py`)

**Purpose:** Detect BIOS version and check for updates

**Information Gathered:**
- BIOS/UEFI version
- BIOS vendor (Dell, HP, Lenovo, etc.)
- BIOS release date
- Motherboard model
- Latest available BIOS version (from vendor)
- Security patches in latest version
- Update urgency (Low/Medium/High)

**Platform Support:**
- **Windows:** WMI queries (`Win32_BIOS`)
- **macOS:** `system_profiler SPHardwareDataType`
- **Linux:** `/sys/class/dmi/id/` files, `dmidecode`

**Implementation:**

```python
class BIOSAuditor(BaseAuditor):
    """Audit BIOS/UEFI version and updates."""

    def audit(self) -> dict:
        """Run BIOS audit."""
        if not self.can_run():
            return {"installed": False, "reason": "Requires admin privileges"}

        result = {
            "installed": True,
            "vendor": self._get_vendor(),
            "version": self._get_version(),
            "release_date": self._get_release_date(),
            "motherboard": self._get_motherboard(),
            "latest_version": self._check_latest_version(),
            "update_available": False,
            "risk_level": "low",
            "educational_content": self._get_educational_content()
        }

        # Determine if update is available
        if result["latest_version"] and result["latest_version"] != result["version"]:
            result["update_available"] = True
            result["risk_level"] = self._assess_risk(result)

        return result

    def _get_vendor(self) -> str:
        """Get BIOS vendor."""
        if platform.system() == "Windows":
            return self._windows_wmi_query("SELECT Manufacturer FROM Win32_BIOS")
        elif platform.system() == "Darwin":
            return self._macos_system_profiler("Model Name")
        elif platform.system() == "Linux":
            return self._linux_read_file("/sys/class/dmi/id/bios_vendor")

    def _get_version(self) -> str:
        """Get BIOS version."""
        # Platform-specific implementation

    def _check_latest_version(self) -> Optional[str]:
        """Check vendor website for latest BIOS."""
        # This is challenging - may require web scraping or APIs
        # For v0.3.0, return None and mark as "manual check required"
        # For v0.4.0, implement vendor-specific checkers
        return None

    def _assess_risk(self, result: dict) -> str:
        """Assess risk level of outdated BIOS."""
        # Check BIOS age
        age_days = (datetime.now() - result["release_date"]).days

        if age_days > 730:  # 2 years
            return "medium"
        elif age_days > 365:  # 1 year
            return "low"
        else:
            return "low"

    def _get_educational_content(self) -> dict:
        """Load educational content about BIOS."""
        return {
            "what_is_it": "BIOS (Basic Input/Output System) is firmware that initializes your hardware during boot.",
            "why_update": "Updates fix bugs, improve hardware compatibility, and patch security vulnerabilities.",
            "when_to_skip": "If your system is stable and you're not experiencing issues, BIOS updates are optional.",
            "how_to_update": "Visit your manufacturer's support website and follow their specific BIOS update tool.",
            "risks": "BIOS updates can brick your system if interrupted. Always use UPS backup power.",
            "learn_more_url": "https://github.com/aramantos/devaudit/blob/main/docs/concepts/bios.md"
        }
```

**Educational Content (`docs/concepts/bios.md`):**
- What is BIOS/UEFI?
- Legacy BIOS vs. Modern UEFI
- Why BIOS updates matter
- When to update (and when not to)
- How to update safely (step-by-step by vendor)
- Risks of updating
- Troubleshooting failed updates

### 2. OS Update Auditor (`os_audit.py`)

**Purpose:** Check for operating system updates and security patches

**Information Gathered:**
- OS version and build number
- Pending updates (count and details)
- Security updates available
- Feature updates available
- Last update check timestamp
- Auto-update status (enabled/disabled)
- Update urgency (Low/Medium/High/Critical)

**Platform Support:**
- **Windows:** `Windows Update API`, PowerShell `Get-WindowsUpdate`
- **macOS:** `softwareupdate` command
- **Linux:** Package manager queries (`apt`, `yum`, `pacman`)

**Implementation:**

```python
class OSAuditor(BaseAuditor):
    """Audit OS update status."""

    def audit(self) -> dict:
        """Run OS audit."""
        result = {
            "os_name": platform.system(),
            "os_version": platform.version(),
            "os_release": platform.release(),
            "pending_updates": [],
            "security_updates": [],
            "feature_updates": [],
            "last_check": None,
            "auto_update_enabled": False,
            "risk_level": "low",
            "educational_content": self._get_educational_content()
        }

        if platform.system() == "Windows":
            result.update(self._audit_windows())
        elif platform.system() == "Darwin":
            result.update(self._audit_macos())
        elif platform.system() == "Linux":
            result.update(self._audit_linux())

        result["risk_level"] = self._assess_risk(result)
        return result

    def _audit_windows(self) -> dict:
        """Audit Windows updates."""
        try:
            # Use PowerShell Get-WindowsUpdate
            cmd = ["powershell", "-Command", "Get-WindowsUpdate | ConvertTo-Json"]
            output = subprocess.check_output(cmd, text=True)
            updates = json.loads(output)

            pending = []
            security = []

            for update in updates:
                update_info = {
                    "title": update["Title"],
                    "kb": update.get("KB", ""),
                    "size": update.get("Size", 0),
                    "is_security": "Security" in update["Title"]
                }
                pending.append(update_info)
                if update_info["is_security"]:
                    security.append(update_info)

            return {
                "pending_updates": pending,
                "security_updates": security,
                "auto_update_enabled": self._check_windows_auto_update()
            }
        except Exception as e:
            return {"error": str(e), "pending_updates": [], "security_updates": []}

    def _audit_macos(self) -> dict:
        """Audit macOS updates."""
        try:
            cmd = ["softwareupdate", "--list"]
            output = subprocess.check_output(cmd, text=True)

            # Parse softwareupdate output
            pending = self._parse_macos_updates(output)

            return {
                "pending_updates": pending,
                "security_updates": [u for u in pending if "security" in u["title"].lower()],
                "auto_update_enabled": self._check_macos_auto_update()
            }
        except Exception as e:
            return {"error": str(e), "pending_updates": [], "security_updates": []}

    def _audit_linux(self) -> dict:
        """Audit Linux updates."""
        # Detect package manager
        if shutil.which("apt"):
            return self._audit_apt()
        elif shutil.which("yum"):
            return self._audit_yum()
        elif shutil.which("pacman"):
            return self._audit_pacman()
        else:
            return {"error": "Unsupported package manager"}

    def _assess_risk(self, result: dict) -> str:
        """Assess risk level."""
        if result["security_updates"]:
            return "high"
        elif len(result["pending_updates"]) > 10:
            return "medium"
        elif result["pending_updates"]:
            return "low"
        else:
            return "none"

    def _get_educational_content(self) -> dict:
        """Load educational content about OS updates."""
        return {
            "what_is_it": "Operating system updates include security patches, bug fixes, and new features.",
            "why_update": "Updates fix known security vulnerabilities that attackers actively exploit.",
            "when_to_skip": "Feature updates can be deferred, but security updates should be applied quickly.",
            "how_to_update": "Use Windows Update, macOS Software Update, or your Linux package manager.",
            "risks": "Updates rarely cause issues, but backup important data before major feature updates.",
            "learn_more_url": "https://github.com/aramantos/devaudit/blob/main/docs/concepts/os_updates.md"
        }
```

### 3. Antivirus Status Auditor (`antivirus_audit.py`)

**Purpose:** Check antivirus installation and status

**Information Gathered:**
- Antivirus installed (yes/no)
- Antivirus name/product
- Real-time protection enabled
- Definition version and age
- Last scan timestamp
- Quarantine item count
- Windows Defender status (Windows)
- Risk level

**Platform Support:**
- **Windows:** WMI `SecurityCenter2`, Windows Defender API
- **macOS:** XProtect status, third-party AV detection
- **Linux:** ClamAV, third-party AV detection

**Implementation:**

```python
class AntivirusAuditor(BaseAuditor):
    """Audit antivirus status."""

    def audit(self) -> dict:
        """Run antivirus audit."""
        result = {
            "antivirus_installed": False,
            "products": [],
            "real_time_protection": False,
            "definition_age_days": None,
            "last_scan": None,
            "risk_level": "high",
            "educational_content": self._get_educational_content()
        }

        if platform.system() == "Windows":
            result.update(self._audit_windows())
        elif platform.system() == "Darwin":
            result.update(self._audit_macos())
        elif platform.system() == "Linux":
            result.update(self._audit_linux())

        result["risk_level"] = self._assess_risk(result)
        return result

    def _audit_windows(self) -> dict:
        """Audit Windows Defender and installed AV."""
        products = []

        # Check Windows Defender
        try:
            cmd = ["powershell", "-Command",
                   "Get-MpComputerStatus | ConvertTo-Json"]
            output = subprocess.check_output(cmd, text=True)
            defender = json.loads(output)

            products.append({
                "name": "Windows Defender",
                "enabled": defender["RealTimeProtectionEnabled"],
                "definition_age": defender["AntivirusSignatureAge"],
                "last_scan": defender.get("LastFullScanEndTime")
            })
        except Exception as e:
            pass

        # Check third-party AV via SecurityCenter2
        try:
            # WMI query for installed AV products
            pass
        except Exception as e:
            pass

        return {
            "antivirus_installed": len(products) > 0,
            "products": products,
            "real_time_protection": any(p["enabled"] for p in products)
        }

    def _assess_risk(self, result: dict) -> str:
        """Assess antivirus risk."""
        if not result["antivirus_installed"]:
            return "high"
        elif not result["real_time_protection"]:
            return "high"
        elif result["definition_age_days"] and result["definition_age_days"] > 7:
            return "medium"
        else:
            return "low"
```

## API Changes

### Enhanced BaseAuditor

```python
class BaseAuditor:
    """Base class for all auditors."""

    @abstractmethod
    def audit(self) -> dict:
        """Run audit and return results."""
        pass

    def get_educational_content(self) -> dict:
        """Return educational content."""
        return {}

    def assess_risk(self, result: dict) -> str:
        """Assess risk level: none, low, medium, high, critical."""
        return "low"

    def can_run(self) -> bool:
        """Check if auditor can run (permissions, platform, etc.)."""
        return True

    def requires_admin(self) -> bool:
        """Check if auditor requires admin/root privileges."""
        return False
```

### New CLI Commands

```bash
# Run all audits (packages + system)
devaudit scan

# Only system audits
devaudit scan --system-only

# Only package audits (existing behavior)
devaudit scan --packages-only

# Specific system auditor
devaudit scan --bios
devaudit scan --os
devaudit scan --antivirus

# Educational mode (verbose explanations)
devaudit scan --educational
```

## Open Questions

1. **BIOS Latest Version Detection**: How to reliably check for latest BIOS without web scraping?
   - **Answer:** v0.3.0 will show current version, mark "manual check required" for latest
   - **Future:** v0.4.0 can add vendor-specific checkers

2. **Admin Privileges**: Should auditors require admin/root?
   - **Answer:** Auditors should gracefully degrade. Show what's possible without admin, note limitations.

3. **Performance**: Will 8 new auditors slow down scans?
   - **Answer:** Run auditors in parallel where possible. Target <60 seconds total.

4. **Educational Content**: Store in markdown files or database?
   - **Answer:** Markdown files in `docs/concepts/` - easier to maintain and version control.
